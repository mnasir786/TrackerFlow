# TrackFlow Desktop Agent

Download the latest TrackFlow desktop agent for your platform.

## Download

| Platform | File |
|----------|------|
| macOS — Apple Silicon (M1/M2/M3) | `TrackFlow-v1.0.19-arm64.dmg` |
| macOS — Intel | `TrackFlow-v1.0.19.dmg` |
| Windows | `TrackFlow-v1.0.19-x64.zip` |

Not sure which Mac you have? Apple menu → About This Mac → look for "Apple M" (Silicon) or "Intel" in the chip/processor line.

Find all releases on the [Releases page](https://github.com/mnasir786/TrackerFlow/releases/latest).

## Installation

**macOS:** Download the correct `.dmg` for your chip, open it, drag TrackFlow to Applications, launch it, and sign in with your account credentials.

**Windows:** Download the `.zip`, **extract it first**, then run `TrackFlow Setup.exe` from the extracted folder.
If Windows shows a SmartScreen warning, click **More info → Run anyway** — this is expected for new apps and the installer is safe.

## About

TrackFlow is a desktop time tracking agent. This repository contains official release binaries only.
Source code is maintained privately.
