# TrackFlow Desktop Agent

Download the latest version from the [Releases page](https://github.com/mnasir786/TrackerFlow/releases/latest).

## Which file to download?

| Platform | File |
|----------|------|
| macOS Apple Silicon (M1/M2/M3) | `TrackFlow-v1.0.38-arm64.dmg` |
| macOS Intel | `TrackFlow-v1.0.38.dmg` |
| Windows | `TrackFlow-v1.0.38-x64.zip` |

Not sure which Mac you have? Click **Apple menu > About This Mac** and look for "Apple M" (Silicon) or "Intel" in the chip line.

## macOS Installation

1. Download the correct `.dmg` for your chip (arm64 = Apple Silicon, no suffix = Intel)
2. Open the `.dmg` file
3. Drag **TrackFlow** to **Applications**
4. Launch TrackFlow from Applications
5. Sign in with the credentials sent by your administrator

> If macOS says the app cannot be opened: go to **System Settings > Privacy and Security** and click **Open Anyway**

## Windows Installation

1. Download `TrackFlow-v1.0.38-x64.zip`
2. **Right-click the zip > Extract All** into a folder — do NOT run the `.exe` directly from inside the zip
3. Open the extracted folder
4. Run `TrackFlow Setup.exe`
5. If Windows SmartScreen appears, click **More info > Run anyway** — this is expected for new apps
6. Sign in with the credentials sent by your administrator

> **Getting "ffmpeg.dll was not found"?** This happens when you run the installer from inside the zip without extracting first. Extract the zip fully, then run the installer.
